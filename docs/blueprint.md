# **App Name**: AvatarTalk

## Core Features:

- Photo Upload: Allow users to upload a photo for the speaking avatar.
- Audio Upload: Allow users to upload an audio file for the avatar to speak.
- AI Lip Sync Generation: Use AI to analyze the uploaded audio and generate lip sync data for the avatar. The tool will adjust the mouth movements.
- Avatar Animation: Animate the avatar using the generated lip sync data so it appears to be speaking the uploaded audio.
- Video Export: Allow users to export the animated avatar as a video file.

## Style Guidelines:

- Primary color: A muted blue (#6699CC) to convey trustworthiness and clarity.
- Background color: Light gray (#F0F0F0), very desaturated, providing a clean backdrop.
- Accent color: A warm orange (#E67E22), which is analogous to the blue, creating contrast and highlighting key actions.
- Font: 'Inter', a sans-serif font, will be used throughout the application to provide a clean and modern look for both headlines and body text.
- Use simple, outline-style icons for intuitive navigation.
- Clean and straightforward layout with a focus on user-friendliness.
- Subtle animations for transitions and feedback, ensuring a smooth user experience.
